# URL Shortener

A simple URL shortening app built with **PHP**, **MySQL**, and **Bootstrap**.

---

## Features
- Shorten long URLs  
- Redirect to original links  
- Click count tracking  

---

## Tech Stack
- PHP  
- MySQL  
- Bootstrap  

---

## Setup
1. Clone the repo  
2. Create a MySQL database  
3. Update DB details in `config.php`  
4. Run on a local server (XAMPP)
